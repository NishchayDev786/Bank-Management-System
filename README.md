# Bank Management System

## 📌 Description
This project simulates basic banking operations using Java and MySQL with JDBC connectivity. It allows users to perform core banking tasks like creating accounts, depositing and withdrawing money, checking balance, and transferring funds between accounts.

## 🚀 Features
- Create Account
- Deposit Money
- Withdraw Money
- Check Balance
- Transfer Money Between Accounts

## 🛠️ Technologies Used
- Java
- JDBC
- MySQL

## 📚 Concepts Covered
- OOP (Object-Oriented Programming)
- Database Connectivity
- Transactions (ACID properties)
- PreparedStatement

## ▶️ How to Run
1. Setup MySQL database (bankdb)
2. Create `accounts` table
3. Update username & password in `DatabaseConnection.java`
4. Run `BankApp.java`
